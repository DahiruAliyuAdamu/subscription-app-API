To start from afresh
delete all bin, public and routes folders
in app.js delete all the content

install nodemon for auto reruning the server when changes made
npm install --save-dev nodemon

install eslint for format and syntax checking
npx eslint --init


in package.json
right below "private": true and before "scripts": {...}
add "type": "module" for using import instead of require
inside script object update "start" keyword to "start": "node app.js" and create "dev": "nodemon app.js"

install dotenv for environment variables
npm install dotenv
